Instruction pour mettre en place le jeu :
- Pour simuler notre application, il faudra installer Solar2D et décompresser notre dossier L3Games.
- Après l'installation, lancer Solar2D Simulator, ouvrir le dossier L3Games décompressé et chercher le fichier main.lua
- Le jeu devrait en théorie fonctionner

Instruction pour mettre en place la BDD :
- Nous utilisons MySQL, il faut créer une db nommée "game_db".
- Le fichier sql.txt dans le dossier à décompresser "Serveur" contient la requête pour construire la table des utilisateurs.

Instruction pour le serveur :
- Il faudra mettre en place Flask et lancer un serveur local.
- Faire attention à modifier le mdp du fichier "app.py" pour l'accès à la base de données.

Instruction pour l'application mobile :
- Mettre l'apk (dans L3Games-Dossier) sur l'appareil sous Android
- Trouver sur l'appareil l'apk et l'installer.
